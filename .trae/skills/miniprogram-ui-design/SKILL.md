# 小程序UI设计技能

这个技能专门帮助改善Taro小程序的用户界面设计，提供现代化的UI方案和最佳实践。

## 主要功能

### 1. UI组件库推荐
- **Taro UI**: 官方UI组件库，包含丰富的组件
- **Vant Weapp**: 轻量、可靠的移动端组件库
- **WeUI**: 微信官方设计语言的基础样式库

### 2. 设计原则
- **简洁明了**: 避免过度设计，保持界面清爽
- **一致性**: 统一的色彩、字体、间距规范
- **易用性**: 符合用户操作习惯的交互设计
- **响应式**: 适配不同屏幕尺寸

### 3. 配色方案
- **主色调**: 选择品牌色作为视觉焦点
- **辅助色**: 用于按钮、标签等交互元素
- **中性色**: 用于文字、背景、边框
- **状态色**: 成功、警告、错误等状态颜色

### 4. 布局优化
- **栅格系统**: 使用flex布局实现响应式设计
- **间距规范**: 统一的边距和间距标准
- **字体层次**: 清晰的标题、正文、辅助文字层级

## 使用场景

1. 当用户觉得当前UI不够美观时
2. 需要推荐小程序UI组件库时
3. 需要配色和布局建议时
4. 想要优化用户体验时

## 实施步骤

1. 分析当前UI问题
2. 推荐合适的组件库
3. 提供配色方案
4. 优化布局结构
5. 实现交互改进
6. 测试和调整
