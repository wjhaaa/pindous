### 前端刷题小程序（个人项目）

面向前端开发者的面试刷题小程序，通过 DeepSeek API 自动生成 8 大知识体系共 216 道结构化问答题，支持分类学习、收藏标记及进度追踪。

- 设计分批 Prompt 工程方案，将 8 大分类拆分为 2-4 批次逐批生成，通过负样本注入去重、难度分布约束及显式转义规则控制输出质量，单次生成控制在 Token 限制内。

- 处理 LLM 输出格式异常问题，通过 JSON Mode + Temperature=0 约束输出格式，配合 3 次重试机制保证生成成功率，避免正则清洗引入二次错误。

- 基于 Taro Storage Sync 实现离线优先的数据管理方案（学习记录 / 收藏 / 标记 / 进度追踪），零后端依赖，小程序即开即用。

- 技术栈：Taro 3.6 + React 18 + TypeScript + DeepSeek API
